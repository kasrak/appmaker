App Maker

Start Building!

App Maker is a tool for beginners to build simple web applications. It runs in your browser, so there's nothing to install.

About

App Maker was created at the 2014 Waterloo Facebook Hackathon by Alex Holyoke, Kasra Kyanzadeh, and Rohan Das.

We hope to make it easy for beginning coders (especially children!) to use what they already know to create fun apps.

App Maker is open source.
