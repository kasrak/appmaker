App Maker

An IDE for beginners to build web applications. Make your own apps, or edit someone elses!

App Maker was created at the 2014 Waterloo Facebook Hackathon by Alex Holyoke, Kasra Kyanzadeh and Rohan Das. We hope to make it easy for beginning coders (especially children!) to use what they already know to create fun apps.

App Maker is now open source; contribute!
